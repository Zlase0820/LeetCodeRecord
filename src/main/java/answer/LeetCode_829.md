[题目传送门](https://leetcode.cn/problems/consecutive-numbers-sum/)




解题思路：
  
  
整除法，类推分析  

被除数(奇)  /  除数(奇)  余0，可以构成  
被除数(奇)  /  除数(偶)  余除数的一半，可以构成  
被除数(偶)  /  除数(奇)  余0，可以构成   
被除数(偶)  /  除数(偶)  余除数的一半，可以构成    
  
  
剪枝，任意一个结果，减去半个除数小于等于0，则停止  
如：  
15，在除到5时，值为3，余数为0  ，此时 3-5/2=1 满足条件  
15，在除到6时，值为2，余数为3  ，此时 2-6/2=-1 不满足条件，剪枝
